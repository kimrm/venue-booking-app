# Holidation

## A venue booking app

This is a booking app made for a student project exam. It is still work in progress.

## Stack

- Next.js
- TypeScript
- Tailwind
- Postgres

## Packages and libraries

- SWR
- Zod
- Framer Motion
- Playwright
- Jest
- react-datepicker
- vercel/postgres
- mapbox
